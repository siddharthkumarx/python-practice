# python-practice

Practice code from working through Python fundamentals (Shradha Khapra / Apna College course), as part of my transition into Data Engineering.

Each file corresponds to one lecture's practice problems, written and run after finishing that lecture's video and revision notes.

## Structure

```
python-practice/
  lecture01_basics.py
  lecture02_strings.py
  lecture03_lists_tuples.py
  lecture04_dicts_sets.py
  lecture05_loops.py
  lecture06_functions_recursion.py
  lecture07_file_io.py
  lecture08_oop_basics.py
  lecture09_oop_advanced.py
```

## Convention

- One file per lecture, named `lectureNN_topic.py`.
- Each file contains the practice problems for that lecture, solved independently (no copy-pasted solutions).
- Comments mark each problem (`# Problem 1: ...`) so the file reads top to bottom in the same order as the lecture.

## Progress

Lectures 1–9 complete: variables/operators/I/O, strings, conditionals, lists & tuples, dictionaries & sets, loops, functions & recursion, file I/O, OOP (basics through inheritance, polymorphism, and operator overloading).

This repo is updated as each new lecture is completed.
